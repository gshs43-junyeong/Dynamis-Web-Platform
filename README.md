# Dynamis-Web-Platform
경남과학고등학교 Dynamis (동아리) 공식 홈페이지입니다. 이 홈페이지에서는 부원들이 자신의 학술적인 자료와 견해를 공유하고 서로 간의 학술적인 교류를 통해 발전하고, 이러한 분위기를 만들고자 하는 것에서 시작합니다. This is the official website of Dynamis from GSHS. This website began with the purpose of creating an environment where members can share their academic resources and perspectives, and grow together through academic exchange.
